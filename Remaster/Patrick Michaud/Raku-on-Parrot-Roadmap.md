# Raku on Parrot Roadmap
    
*Originally published on [26 November 2007](https://use-perl.github.io/user/pmichaud/journal/34973/) by Patrick Michaud.*

On Friday I wrote up a brief ROADMAP for Raku on Parrot development, and rather than repeat it all as a journal post I'll just link to it.  It really does describe where we've been, and where we're going in the next couple of months as far as Raku development.

If you're wondering where this fits -- right now I'm aware of three Raku implementations underway:

- Pugs - Haskell-based implementation (http://www.pugscode.org/)
- Raku - Parrot-based implementation (http://www.parrotcode.org/)
- KindaPerl6 - Perl/6-based implementation (http://www.pugscode.org/)

So, the implementation I've been focusing on is the one for Parrot, and that's where my journal entries will tend to focus.
